A plugin for embedding Archidekt decks and data into a WordPress site.

== Description ==

This plugin provides a shortcode for embedding a deck from Archidekt.

== Installation ==

1. Upload to the `/wp-content/plugins/` directory
1. Activate the plugin
1. Visit Settings > Archidekt Settings and configure to meet your needs
